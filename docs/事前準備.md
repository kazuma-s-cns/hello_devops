# 事前準備

## Step 1: GitHub アカウントの作成

作成していない場合は、GitHub アカウントを作成してください。

## Step 2: リポジトリのセットアップ

1. このリポジトリを自分の GitHub アカウントに **Fork** する
2. Fork したリポジトリの **Settings > Pages** を開き、Source を「**GitHub Actions**」に設定する

## Step 3: GitHub Codespaces の起動

1. リポジトリのトップページで `Code` ボタンをクリック
2. `Codespaces` タブを選択
3. `Create codespace on main` をクリック
4. Codespace が起動したら、ターミナルで以下を実行して動作確認する

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

> [!NOTE]
> `npm run dev` を実行すると、Codespaces がポートフォワーディングのポップアップを表示します。「ブラウザで開く」をクリックして電卓アプリが表示されることを確認してください。
